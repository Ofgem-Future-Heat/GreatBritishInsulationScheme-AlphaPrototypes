Requesting for an account

If you want to create an account on the ECO Plus service, you can request for an account to be created in the following ways:

Request for an account as an authorised user

In order to request an account, you will need to be authorised. As an authorised user, once your account has been created and validated, you will be able to add additional users.

The requirements to become an authorised user are as follows:

  • Complete the authorised user request form
  • Provide additional mandatory information
  • Send the completed form and additional mandatory inform to request@eco-plus-accounts.gov.uk
