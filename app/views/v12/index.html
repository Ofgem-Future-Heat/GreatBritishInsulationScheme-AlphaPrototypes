The Great British Insulation Scheme portal

 

Energy suppliers are obligated to let Ofgem know about measures they've delivered to customers. To do this, they use the Great British Insulation Scheme portal to record these measures.

All users need a GOV.UK One Login account and a Great British Insulation Scheme user profile to access and use the portal.

Accessing the portal

To do this you will need a:

  • GOV.UK One Login account
  • Great British Insulation Scheme user profile

If you already have both

If you have a GOV.UK One Login account and a Great British Insulation Scheme user profile, you're ready to use the portal. You will be taken the GOV.UK One Login website, where you will login. Then you will be taken to the portal

Use the Great British Insulation Scheme portal

Why you need a GOV.UK One Login account

We use your GOV.UK One Login account to verify you and create your Great British Insulation Scheme user profile.

You should create your GOV.UK One Login using your work email address.

Creating a GOV.UK One Login account

You create this account on the GOV.UK One Login website. You will receive step by step instructions on how to do this.

Create a GOV.UK One Login account

Great British Insulation Scheme user profile

Once you have a GOV.UK One Login account, you're ready to request a Great British Insulation Scheme user profile from Ofgem.

Use the same email address as the one used to create your GOV.UK One Login. This allows us to verify your identity. This should be your work email address.

Request for user profiles must come from authorised signatory

Only the authorised signatory for your organisation can request Great British Insulation Scheme user profiles on behalf of colleagues. You can make these requests by email at GBIS.enquiry@ofgem.gov.uk

Include your 'authorised data provider statement of authorisation form'

In the email to Ofgem, you must attach your 'authorised data provider statement of authorisation form'. This is the document a supplier uses to let Ofgem know the details of the person that is responsible for requesting user profiles to use the portal

Download the authorised data provider form (42.82 KB)

The people requesting user profiles must use the same email address as they have when creating a GOV.UK One Login account for verification reasons.

How long will this take to set up?

After sending your request, we will contact you by email about creating user profiles. If you already have a GOV.UK One Login account and you respond to our email reply immediately, we should be able to give you access to the portal promptly.

Use the Great British Insulation Scheme portal

Once you have the required Great British Insulation Scheme profile and GOV.UK One Login account, you can use the portal.

The portal can be used to:

  • upload measures
  • search for measures
  • view error reports

Documents you need to use with the Great British Insulation Scheme

 

There are documents you need when using the Great British Insulation Scheme. They are the:

  • Measure notification template
  • Data dictionary

Measure notification template

This is the document a supplier uses to add the information about installed measures. They then upload this document to the portal

Download the measure notification template

Data dictionary

This document includes information to help suppliers complete the measure notification template.

It contains a:

  • description of every field on the template
  • acceptable data that may be entered in those fields
  • circumstances under which this data should be included

Suppliers should only use the most recent version of the dictionary and template unless instructed otherwise. The information in this document relates to all measures installed by the supplier from 30 March 2023.

Download the data dictionary

If you need help accessing or using the portal

 

You can contact us by email at GBIS.enquiry@ofgem.gov.uk if you need help or have questions about:

  • getting access to the portal
  • using the portal
  • anything else relating to the Great British Insulation Scheme
