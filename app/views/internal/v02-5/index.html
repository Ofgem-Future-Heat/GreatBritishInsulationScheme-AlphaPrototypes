Latest announcement 12/10/2023

ECO4.24.01 Planned Release

Please be aware that this release has been cancelled due to an issue with one of our release tools we use to ensure that our changes do not impact the reliability and resilience of the register. Changes outlined below for this release will be rolled into our January release unless stated otherwise. Apologies for any inconveniences.

The ECO Register will be offline on Tuesday 16th January 2024 from 17:00 to 19:00 due to a planned release.

Functionality being deployed is as follows:

You can buy:

  • Late Automatic Late Rules and Report Update
  • T4110: Observed Value will be Project Status name instead of numeric status ID
  • ECO3 interim Completed Measure Transfers
  • Innovation Measure Numbers 022 and 023
  • Removal of "Town" field checks with TrustMark
  • "Social Housing" Tenure Type applicable for "in-fill" elegibility type projects (P40517, P40518, P40519)
  • Update to the Partial Project Score matrix to published version 5

Live issue fixes being deployed are:

  • Clear Partial Project Scores associated with Exemption Measures
  • TP0005 will now be case insensitive
  • Fixed an issue where large measure uploads with MRNs of 11-characters were occasionally timing out
  • We have also added some background processing to reduce the build up of measures getting stuck in "Measure Awaiting Verification" status
