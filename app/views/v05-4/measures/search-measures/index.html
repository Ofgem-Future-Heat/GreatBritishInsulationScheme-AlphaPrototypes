Searching the measure register

You can search and view the measures you've previously uploaded to the Great British Insulation Scheme portal.

Different ways to search for measures

You can search for measures in two ways:

  • Using a single measure reference number.
  • Uploading a file with multiple measure reference numbers.

Single measure reference numbers

You can search for individual measures using a single measure reference number.

To do so:

  • Find the measure reference number from the original project notification file you uploaded.
  • Copy the measure reference number.
  • Add it to the measure reference number field on the next page.
  • Select the "Search" button.

Multiple measure reference numbers in a file

To search for measures using a measure reference number file, you should:

  • Create a CSV file in Microsoft Excel.
  • Include a single column with only the measure reference numbers you want to search for.
  • Save the CSV file.
  • Go to the search page in the Great British Insulation Scheme.
  • Select "Choose file" and select the CSV file you created.
  • Select "Upload measure reference file" .
  • Select the "Search" button.
