Existing error reports

All existing error reports are listed here by what stage they're at in the process of validation.

The list of error reports includes all reports generated as a result of you uploading a notification template. The most recent reports are shown first.

! Warning Error reports can contain up to 15,000 errors. Where reports have this many errors, it shows there is possibly an issue with the data uploaded originally. You should review the measure notification file uploaded originally and ensure the data is accurate. If the problem persists, email Ofgem at GBIS.enquiry@ofgem.gov.uk.

You should download any relevant error report then follow the next steps.
